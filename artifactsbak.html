Artifacts - Alexander Pittman

Graphics Card Removal & Install Description: This artifact details the removal and reinstallation of a dedicated video card from a donor Xeon-based system and reinstalling it into a newer i3 Intel PC.
Selection Rationale: This artifact was selected because I enjoy working on computer hardware and one of my sibling's friends needed to have this work performed as part of an upgrade to a newer desktop computer.
Competencies Demonstrated: This demonstrates my ability to perform IT support and PC hardware upgrade tasks. This effort is an activity that was similar to a virtual lab completed during my Troubleshooting Applications and Hardware class at Owens College, but as an actual hands-on exercise.
Expected Outcome: As a result of this artifact presentation, the destination PC will have a new graphics card and its associated drivers installed, followed by a successful test render.
Format: Presentation with video, audio, and text
Google Cloud VM Instantiation Description: This artifact involves the instantiation of a virtual machine (VM) on Google Cloud Platform (GCP) that is capable of running a specific application. To achieve this, the VM specifications are defined at creation time to support the processor and memory requirements of the intended application (Minecraft).
Selection Rationale: This artifact was chosen because hyperscalar platforms are prevalent in enterprise environments, and proficiency in their basic operation and administration is often required for employment.
Competencies Demonstrated: This artifact demonstrates my ability to log in to, administer, and configure basic functions on a hyperscalar system.
Expected Outcome: As a result of this artifact presentation, a VM will be created, an OS installed, and remote access setup, which will allow inbound connections to a yet to be deployed application.
Format: Presentation with images, audio, and text
Application Installation on a VM Description: This artifact shows how to install an application on an existing virtual machine running Debian Linux on Google Cloud Platform (GCP). It involves downloading both a required Java runtime and the intended application, Minecraft. Once downloaded, the target application is then setup and tested to verify it runs correctly via verification of a remote client login.
Selection Rationale: This artifact was chosen because a significant amount of my personal time is spent working on code and game development. This evidences my ability to deploy a software package to be used in an IT environment and configure it at a low level for use.
Competencies Demonstrated: This artifact showcases my ability to take an application, gather its installation requireents, deploy it in live environment, and then confirm the ability for users to properly connect.
Expected Outcome: As a result of this artifact presentation, an application server running Minecraft code will be successfully installed and enable a client to properly connect and use its services.
Format: Webpage with images and text
