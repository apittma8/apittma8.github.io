Artifacts - Google Cloud VM Instantiation

Click here to view this demonstration Description: This artifact involves the instantiation of a virtual machine (VM) on Google Cloud Platform (GCP) that is capable of running a specific application. To achieve this, the VM specifications are defined at creation time to support the processor and memory requirements of the intended application (Minecraft).
Selection Rationale: This artifact was chosen because hyperscalar platforms are prevalent in enterprise environments, and proficiency in their basic operation and administration is often required for employment.
Competencies Demonstrated: This artifact demonstrates my ability to log in to, administer, and configure basic functions on a hyperscalar system.
Expected Outcome: As a result of this artifact presentation, a VM will be created, an OS installed, and remote access setup, which will allow inbound connections to a yet to be deployed application.
Format: Presentation with images, audio, and text
